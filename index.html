Gcovr

The gcovr command provides a utility for managing the use of the GNU gcov utility and generating summarized code coverage results. This command is inspired by the Python coverage.py package, which provides a similar utility in Python. gcovr produces either compact human-readable summary reports or machine readable XML reports.
